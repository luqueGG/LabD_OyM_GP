# Propuestas de Mejora

## Propuesta Individual

Después de revisar el proyecto, propongo las siguientes mejoras:

1. Implementar indicadores de desempeño para cada proceso.
2. Automatizar el escaneo de vulnerabilidades de imágenes Docker mediante integración continua.
3. Incorporar políticas de respaldo y recuperación ante desastres.
4. Definir niveles de acceso mediante roles para estudiantes, docentes y administradores.
5. Implementar un procedimiento formal para aprobación de nuevas imágenes.
6. Mantener un historial de versiones de imágenes Docker.
7. Incorporar métricas de utilización de laboratorios.
8. Centralizar la documentación técnica.

---

# Síntesis del Equipo

Como síntesis de las propuestas del grupo se recomienda:

- Estandarizar todos los procesos del laboratorio.
- Automatizar la gestión de imágenes Docker.
- Definir claramente los responsables de cada proceso.
- Implementar indicadores para evaluar el desempeño.
- Fortalecer la trazabilidad mediante control de versiones.
- Mejorar la documentación técnica y operativa.
